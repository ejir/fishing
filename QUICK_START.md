# 快速开始指南 🚀

## 安装与运行

### 1. 安装依赖
```bash
npm install
```

### 2. 启动游戏（推荐）
```bash
npm run dev
```
这将同时启动：
- 前端开发服务器: http://localhost:5173
- 后端Socket.IO服务器: http://localhost:3000

浏览器会自动打开游戏页面。

## 创建你的角色 🎨

### 基础操作
1. **选择图元类型**: 点击圆形、矩形、三角形或椭圆按钮
2. **设置颜色**: 点击颜色选择器选择你喜欢的颜色
3. **调整大小**: 输入像素值 (10-80)
4. **设置旋转**: 输入角度 (0-360度)
5. **添加图元**: 点击"添加图元"按钮

### 拖拽编辑（核心功能）✨
在预览画布上直接操作：

- **🖱️ 拖拽移动**: 
  - 用鼠标左键点击并拖拽图元
  - 图元会跟随鼠标移动
  - 自动限制在编辑区域内

- **🔍 Shift + 拖拽缩放**:
  - 按住 `Shift` 键
  - 拖拽鼠标改变图元大小
  - 越远越大，越近越小

- **🔄 Ctrl + 拖拽旋转**:
  - 按住 `Ctrl` 键 (Mac上是 `Cmd`)
  - 拖拽鼠标旋转图元
  - 围绕图元中心旋转

- **🗑️ 删除图元**:
  - 点击选中图元（会高亮显示）
  - 按 `Delete` 键删除
  - 或点击列表中的删除按钮

### 预览和调整
- 画布中心有十字参考线
- 虚线圆圈显示推荐编辑范围
- 选中的图元会以蓝色边框高亮
- 实时查看编辑效果

## 开始游戏 🎮

### 进入游戏
点击 **"🚀 开始游戏"** 按钮进入多人世界

### 游戏控制
- **W/A/S/D**: 移动你的角色
- **鼠标移动**: 控制角色朝向
- **Enter**: 发送聊天消息

### 游戏目标
- 💔 靠近其他玩家增加节操值
- 📊 节操值越高，体型越大，但速度越慢
- 🏆 登上节操排行榜榜首

## 编辑器高级技巧 💡

### 创建复杂角色
1. **头部**: 使用大圆形作为基础
2. **眼睛**: 添加两个小圆形
3. **身体**: 使用矩形或椭圆
4. **装饰**: 用三角形制作帽子或其他装饰

### 对称设计
- 手动计算X坐标实现左右对称
- 例如：左眼 x=-20，右眼 x=20

### 图层管理
- 后添加的图元会显示在前面
- 通过列表查看所有图元
- 点击列表项选中对应图元

## 多人游戏技巧 🎯

### 增长策略
- 寻找其他玩家密集的区域
- 与多个玩家重叠可快速增长
- 节操值 = 重叠玩家数 × 2/秒

### 生存策略
- 体型过大会严重降低速度
- 适时远离人群降低节操值
- 平衡体型和速度找到最优策略

### 社交互动
- 使用聊天系统与玩家交流
- 协作或竞争随你选择
- 观察排行榜了解高手策略

## 故障排除 🔧

### 端口被占用
如果3000或5173端口被占用：
```bash
# 查找占用进程
lsof -i :3000
lsof -i :5173

# 结束进程
kill -9 <PID>
```

### 连接失败
- 确保服务器正在运行
- 检查浏览器控制台错误信息
- 刷新页面重试

### 编辑器不响应
- 确保在预览画布内操作
- 检查是否正确按下修饰键（Shift/Ctrl）
- 刷新页面重新开始

## 键盘快捷键汇总 ⌨️

### 编辑器
- `Delete` - 删除选中的图元
- `Shift + 拖拽` - 缩放图元
- `Ctrl/Cmd + 拖拽` - 旋转图元

### 游戏
- `W` - 向上移动
- `S` - 向下移动
- `A` - 向左移动
- `D` - 向右移动
- `Enter` - 发送聊天
